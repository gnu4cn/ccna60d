# 网关负载均衡协议


与 HSRP 一样，网关负载均衡协议（GLBP），也是一种思科专有的协议。GLBP 以类似 HSRP 及 VRRP 的方式，提供了高网络可用性。但不同于任何时候都只有一个网关活动转发某个特定分组流量的 HSRP 与 VRRP，GLBP 允许同一 GLBP 分组内的多个网关，于同一时刻都积极转发网络流量。


GLBP 的网关，通过使用 UDP 端口 3222，每 3 秒发送 `Hello` 报文到组播地址 `224.0.0.102` 通信。这在下图 32.24 中得以演示。
