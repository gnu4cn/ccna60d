IPv6 已经开发了好多年，实际上已部署在全世界的网络上（与 IPv4 一起）。许多网络工程师已对必须学习一种新的寻址方法表示恐惧，我（作者）甚至听到许多人说，他们希望在 IPv6 成为要求前就退休。

然而，这种担心是没有根据的。IPv6 是一种用户友好的格式，一旦咱们习惯了他，咱们将发现，他是对 IPv4 的一项改进，而咱们可能会更喜欢他。IPv6 在 CCNA 考试中占了很大比重；因此，咱们需要轻松应对其工作原理的掌握，以及如何配置地址、掌握标准，与应用 IPv6 地址解决网络需求。

今天，咱们将学习以下内容：

- IPv6 的历史
- IPv6 地址分配
- IPv6 地址分配的表示法
- 不同的 IPv6 地址类型
- IPv6 的协议与机制
- IPv6 下的子网划分
- IPv6 与 IPv4 的对比
