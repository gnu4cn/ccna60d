# 以太网供电

以太网供电（PoE）是考试大纲中的另一个新项目，因此咱们可以预料到会被问及这类问题。PoE 是一项允许交换机通过以太网线缆，与数据一起传输电力的技术。咱们将发现其最常用于 IP 电话、监控摄像头与无线接入点等。这些设备被称为 PD 或受电设备。

使用 UTP 线缆中的四根导线（当然不是用于数据传输的那些），减少了办公室布线的必要性，因为每个受电设备都可在无需连接到主电源的单独电源下运行。请注意，每台受电设备都必须连接到交换机；咱们不能通过菊花链（手拉手）连接到另一台受电设备获取电源 <sup>1</sup>。

PoE 只在一些特殊交换机型号上才受支持，他们通常价格较高。在 Cisco 交换机上，PoE 可在接口配置模式下，使用 `power inline` 命令配置（默认情况下这是打开的，但可以使用 `no power inline` 命令禁用）。请关注免费资源页面上 PoE 的实验。


PoE 被定义为 [IEEE 802.3af](https://www.ieee802.org/3/bu/index.html) 中的一项标准。符合 802.3af 并支持 CDP 的 Cisco PD，可以覆盖默认的电源分类。在咱们进行任何配置前，请一定咨询思科公司，否则可能导致保修失效。思科设备同时支持电源使用管理及电源监控。

当某个受监控端口的功耗，超过由管理员配置的最大值时，则以下操作就会发生：

- 发出一条 `syslog` 消息；
- 该受监控端口会被关闭，并被因报错而禁用；
- 已分配的电量会被释放。
