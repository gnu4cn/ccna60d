# 以太网供电


以太网供电（PoE）是考试大纲中的另一个新项目，因此您可以预料到会有这样的问题。PoE 是一种允许交换机通过以太网电缆传输电力和数据的技术。您会发现它最常用于 IP 电话、监控摄像头和无线接入点。这些设备被称为 PD 或受电设备。在 UTP 电缆中使用四根导线（当然不是用于数据传输的导线）可以减少办公室布线的必要性，因为每个受电设备都可以在没有单独电源的情况下运行。请注意，每台受电设备都必须连接到交换机；不能通过菊花链连接到另一台受电设备获取电源。
只有特殊型号的交换机才支持 PoE，这些交换机通常价格较高。在 Cisco 交换机上，可以在接口配置模式下使用 power inline 命令配置 PoE（默认情况下已打开，但可以使用 no power inline 命令禁用）。请关注免费资源页面上的 PoE 实验室。
PoE 在 IEEE 802.3af 中被定义为一种标准。符合 802.3af 并支持 CDP 的 Cisco PD 可以覆盖默认电源分类。在进行任何配置之前，请咨询思科公司，否则可能导致保修失效。思科设备支持电源使用管理和电源监控。
