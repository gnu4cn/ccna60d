增强型内部网关路由协议 (EIGRP)， 是种由 Cisco 开发的专有内部网关协议 (IGP)。EIGRP 包含诸如水平分割的一些传统距离矢量的特征，以及一些类似于诸如增量更新等链路状态路由协议用到的特征。

尽管 EIGRP 有着一些链路状态路由协议的特征，但 EIGRP 属于距离矢量路由协议分类，并且被称为一种高级距离矢量路由协议。EIGRP 使用协议编号 88，直接于 IP 之上。

虽然不可能深入研究所有潜在的 EIGRP 问题情形，但这一教学模组将讨论在 EIGRP 作为内部网关协议 (IGP) 选项实施时，一些最常见的问题情形。


今天，咱们将学习以下内容：


- 思科 EIGRP 概述及基础知识
- EIGRP 配置基础
- EIGRP 的邻居发现和维护
- 度量值、DUAL 与拓扑数据表
- 等成本和不等成本的负载分担
- 使用 EIGRP 的默认路由
- EIGRP 网络中的水平分割
- EIGRP 的路由汇总
- 掌握被动接口
- 掌握 EIGRP 路由器 ID 的使用
- EIGRP 的故障排除
