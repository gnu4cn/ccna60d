欢迎来到许多人都认为是 CCNA 考试大纲中，最难理解的领域之一。为了掌握 CCNA 考试中的 IP 寻址，我们必须介绍一些二进制算术与十六进制编号系统、地址类别、二的幂次与子网零、广播与网络地址，以及计算子网和主机地址的一些公式。

不过请别担心，这是个过程，不是一次性的活动，所以请按照我（作者）的课文做，然后放心，我们将多次回来复习这些概念。

今天，咱们将学习以下内容：

- 网络流量的类型
- IP 寻址
- 地址类别
- 使用 IP 地址
- 子网划分
- 路由汇总
- 可变长度子网掩码
- 解决 IP 寻址问题
